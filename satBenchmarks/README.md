# Наборы для тестирования

Каталоги содержат формулы в формате DIMACS CNF, полученные из
[SATLIB](https://www.cs.ubc.ca/~hoos/SATLIB/benchm.html):

- `uf50-218`, `uf75-325`: выполнимые случайные формулы 3-SAT.
- `uuf50-218`, `uuf75-325`: невыполнимые случайные формулы 3-SAT.
- `flat50-115`: выполнимые формулы, кодирующие раскраску графа.
- `pigeon-hole`: невыполнимые формулы, кодирующие принцип Дирихле.
- `aim`: искусственно сгенерированные выполнимые и невыполнимые формулы.

Наборы включены в репозиторий для воспроизводимости сравнения
производительности. Исходные описания и сведения о распространении доступны на
сайте SATLIB.
