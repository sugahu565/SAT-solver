### Описание 
Данная учебная практика направлена на изучение и реализацию алгоритма DPLL для решения задачи о выполнимости конъюктивно нормальной формы. 

### Запуск

Сборка: 
```bash
make build 
```
Исполняемые файлы лежат в build/bin/

Запуск тестов: 
```bash
make test 
```

Запуск линтера: 
```bash
make lint 
```

Очистка папки build:
```bash
make clean 
```

### Замеры производительности

`measure_all.sh` является единственной точкой входа для утилит замера. Скрипт
проверяет выбор эвристик `naive` и `jwh`, фиксирует частоту CPU на время запуска
и сравнивает обе эвристики на всех `.cnf`-файлах указанного датасета:

```bash
./measure_all.sh satBenchmarks/uf50-218
```

Перед запуском соберите solver командой `make build` и установите `hyperfine`.
Результаты сохраняются в `results/` в форматах JSON и Markdown. Скрипт использует
`sudo` для временного изменения настроек CPU и восстанавливает их при завершении.
